About WordFinder

WordFinder is a dictionary, thesaurus, rhyme dictionary, and near-rhyme dictionary. It was created and is maintained by @calebeby.

WordFinder's results are powered by the Datamuse API. It follows Google's Material Design Guidelines by using @calebeby's own SCSS framework, Sol. WordFinder uses SCSS, Slim, and Coffeescript for code that is easier to read and write. These are compiled with Middleman, a static site generator. The source code for WordFinder is available on GitHub
